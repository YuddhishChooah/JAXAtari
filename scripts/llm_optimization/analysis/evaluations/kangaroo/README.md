# Kangaroo Evaluation Summaries

This folder stores compact Kangaroo evaluation summaries only. It is meant to
replace scattered root-level Kangaroo JSON files and bulky raw run folders.

Most reports should start from:

- `kangaroo_policy_v6_overlap_patch2_retuned_vectorized_128eps_50000steps.json`
- `kangaroo_shaped_v2_post200_guard_eval32_10000steps.json`
- `kangaroo_shaped_v2_post200_guard_dodge_eval32_10000steps.json`
- `kangaroo_dense_reward_20260505_policy_v3_eval32_10000steps.json`

For the human-readable interpretation, use
`../../kangaroo_evidence_index.md` from the `analysis` directory.
